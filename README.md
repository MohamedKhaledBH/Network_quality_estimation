# Network Quality Estimation

Deep Learning project for estimating network quality based on signal characteristics.

## Project Overview

This project implements deep learning models to estimate network quality using deep signal processing techniques. It includes Jupyter notebooks for exploration and an application module for inference.

## Contents

- **app.py** - Main application for network quality estimation
- **deepsig_fixed_notebook.ipynb** - Fixed implementation notebook
- **deepsig_improved_notebook.ipynb** - Improved implementation with enhancements
- **requirements-deploy.txt** - Python dependencies

## Features

- Deep signal processing for network analysis
- Multiple model implementations (fixed and improved versions)
- Jupyter notebook-based experimentation
- Production-ready application code

## Installation

1. Clone the repository:
```bash
git clone https://github.com/MohamedKhaledBH/Network_quality_estimation.git
cd Network_quality_estimation
```

2. Install dependencies:
```bash
pip install -r requirements-deploy.txt
```

## Usage

### Running the Application

```bash
python app.py
```

### Using Jupyter Notebooks

```bash
jupyter notebook deepsig_improved_notebook.ipynb
```
